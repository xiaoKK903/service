# 智能客服前后端项目

这是一个智能客服前后端项目，包含Vue 3客户端、Vue 3人工客服工作台和后端Express API。

## 项目结构

```
智能客服/
├── client/           # Vue 3客户端
│   ├── src/          # 源代码
│   │   ├── App.vue   # 主应用组件
│   │   └── main.js   # 入口文件
│   ├── package.json  # 客户端依赖配置
│   └── vite.config.js # Vite配置
├── admin/            # Vue 3人工客服工作台
│   ├── src/          # 源代码
│   │   ├── App.vue   # 主应用组件
│   │   └── main.js   # 入口文件
│   ├── package.json  # 工作台依赖配置
│   └── vite.config.js # Vite配置
├── backend/          # 后端Express项目
│   ├── index.js      # 主服务器文件
│   └── package.json  # 后端依赖配置
└── README.md         # 项目说明
```

## 技术栈

- **客户端**: Vue 3, Vite, CSS
- **人工客服工作台**: Vue 3, Vite, CSS
- **后端**: Express, Node.js
- **通信**: HTTP POST请求

## 运行步骤

### 1. 启动后端服务器

```bash
# 进入后端目录
cd backend

# 安装依赖
npm install

# 启动服务器
npm start
```

后端服务器将运行在 http://localhost:3001/

### 2. 启动客户端开发服务器 (Vue 3)

```bash
# 进入客户端目录
cd client

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

客户端开发服务器将运行在 http://localhost:5175/ (或其他可用端口)

### 3. 启动人工客服工作台开发服务器 (Vue 3)

```bash
# 进入工作台目录
cd admin

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

人工客服工作台开发服务器将运行在 http://localhost:5174/ (或其他可用端口)

## 使用方法

### 客户端
1. 在客户端页面的输入框中输入消息
2. 点击"发送"按钮
3. 后端会接收到消息并返回响应
4. 客户端会显示后端的响应消息

### 人工客服工作台
1. 在工作台页面的左侧选择客户
2. 在聊天窗口中查看与客户的对话历史
3. 在输入框中输入回复消息
4. 点击"发送"按钮发送消息

## API端点

- **POST /api/hello**: 接收前端发送的消息并返回响应
  - 请求体: `{ "message": "你的消息" }`
  - 响应: `{ "message": "后端收到消息: 你的消息" }`

## 功能说明

- **客户端 (Vue 3)**: 提供用户界面，发送消息到后端并显示响应，包含聊天记录功能
- **人工客服工作台 (Vue 3)**: 提供客服人员使用的界面，显示客户列表和聊天记录
- **后端**: 接收前端消息，处理后返回响应
- **跨域处理**: 使用CORS中间件允许跨域请求
